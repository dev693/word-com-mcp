# word-com-mcp

A C# [Model Context Protocol (MCP)](https://modelcontextprotocol.io) server that drives
Microsoft Word through its COM automation API, exposing Word operations as MCP tools.

## Status

Early scaffolding. This repository currently contains only project setup.

## Requirements

- Windows with Microsoft Word installed (COM automation requires the desktop Word application)
- .NET SDK
